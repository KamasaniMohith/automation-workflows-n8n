# 🏠 Real Estate Leads Capture

> Automatically captures, qualifies, and follows up with real estate leads — on autopilot!

---

## ✨ What It Does

This workflow handles real estate lead management end-to-end:
- Captures leads from web forms or CRM webhooks
- Automatically qualifies leads based on set criteria
- Sends personalized follow-up emails instantly
- Logs all lead data into a Google Sheet or CRM
- Triggers notifications to the sales team in real time

No lead falls through the cracks!

---

## 🛠️ Tools & Tech

| Tool | Purpose |
|------|---------|
| n8n | Workflow automation |
| Webhook | Lead capture trigger |
| Email / Gmail | Auto follow-up |
| Google Sheets | Lead database |
| Slack (optional) | Team notifications |

## 🚀 How to Use

1. Import `workflow.json` into your n8n instance
2. Connect Gmail and Google Sheets credentials
3. Set your webhook URL as the form submission endpoint
4. Customize the email templates and lead criteria
5. Activate and start capturing leads automatically!
