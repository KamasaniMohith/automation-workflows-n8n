# 🔧 Auto Repair Pipeline

> Automates the entire auto repair shop workflow — from job intake to completion!

---

## ✨ What It Does

This workflow streamlines auto repair operations:
- Receives repair job requests automatically
- Assigns jobs to available technicians
- Sends status updates to customers at each stage
- Tracks job progress from intake to completion
- Logs all jobs and outcomes for reporting

Less admin work, more wrenching!

---

## 🛠️ Tools & Tech

| Tool | Purpose |
|------|---------|
| n8n | Workflow automation |
| Webhook | Job intake trigger |
| Email / SMS | Customer updates |
| Google Sheets | Job tracking |

## 🚀 How to Use

1. Import `workflow.json` into your n8n instance
2. Connect your email and Google Sheets credentials
3. Set up the webhook as your job intake endpoint
4. Customize status messages for your shop
5. Activate and let the pipeline run itself!
