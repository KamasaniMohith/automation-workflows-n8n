# 🤖 AI-Powered Booking System

> A fully automated booking system powered by AI — zero backend server required!

---

## ✨ What It Does

This workflow handles the entire booking lifecycle automatically:
- Accepts booking requests via webhook
- Uses AI to validate, process, and confirm bookings
- Sends automated confirmation emails to customers
- Logs all bookings to a database/sheet in real time

No backend server. No manual work. Pure automation.

---

## 🛠️ Tools & Tech

| Tool | Purpose |
|------|---------|
| n8n | Workflow automation |
| AI (LLM) | Smart booking validation |
| Webhook | Receives booking requests |
| Email | Sends confirmations |
| Google Sheets | Booking logs |


## 🚀 How to Use

1. Import `workflow.json` into your n8n instance
2. Set up your credentials (email, Google Sheets, AI API key)
3. Copy the webhook URL and connect it to your booking form
4. Activate the workflow — it runs on autopilot!
