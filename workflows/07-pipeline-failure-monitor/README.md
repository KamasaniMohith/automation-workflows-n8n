# 🚨 Pipeline Failure Monitoring

> Instantly detects when any automation fails and alerts your team — before it becomes a problem!

---

## ✨ What It Does

This workflow acts as a watchdog for all your other automations:
- Monitors n8n workflow executions continuously
- Detects failures and errors in real time
- Sends instant alerts via email or Slack
- Logs all failures with timestamps and error details
- Helps you debug and fix issues fast

Sleep easy knowing your pipelines are being watched 24/7!

---

## 🛠️ Tools & Tech

| Tool | Purpose |
|------|---------|
| n8n | Workflow automation |
| Error Trigger | Catches workflow failures |
| Slack / Email | Instant alerts |
| Google Sheets | Failure log |



## 🚀 How to Use

1. Import `workflow.json` into your n8n instance
2. Connect Slack and/or Gmail credentials
3. Set your alert channel/email in the notification nodes
4. Activate — it monitors all your other workflows automatically!
