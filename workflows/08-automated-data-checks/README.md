# ✅ Automated Data Checks

> Automatically validates, cleans, and verifies your data — no more manual spot-checking!

---

## ✨ What It Does

This workflow runs automated quality checks on your data:
- Pulls data from Google Sheets or a database
- Runs validation rules to catch errors and anomalies
- Flags and highlights bad or missing data
- Sends a clean summary report of all issues found
- Optionally auto-fixes common data problems

Data quality on autopilot!

---

## 🛠️ Tools & Tech

| Tool | Purpose |
|------|---------|
| n8n | Workflow automation |
| Google Sheets | Data source |
| JavaScript (Code node) | Validation logic |
| Email / Slack | Issues report |
| Schedule Trigger | Runs on a set schedule |

---


## 🚀 How to Use

1. Import `workflow.json` into your n8n instance
2. Connect your Google Sheets account
3. Set your Sheet ID and define your validation rules in the Code node
4. Add your alert email or Slack channel
5. Activate — data checks run automatically on schedule!
